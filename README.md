# Тренер ТСПП — Статусы сотрудников

Telegram Mini App для тренеров службы прямых продаж ТСПП. Показывает список сотрудников по статусам: **Проверено**, **Ждут Проверку**, **Исправляют ошибки**.

## Источник данных

- **NocoDB**: https://nocodb.puzzlebot.top/
- Таблицы: `Сотрудники`, `День 1`

## Подключение в PuzzleBot

1. Загрузите `index.html` на хостинг (Vercel, Netlify или свой сервер)
2. В PuzzleBot создайте Web App или ссылку на этот URL
3. Убедитесь, что NocoDB API токен корректный (в `index.html` в константе `API_TOKEN`)

## Требования к таблицам NocoDB

### День 1
- **ФИО сотрудника** (или поле, содержащее «ФИО»)
- **Статус** (или похожее поле) — значения: `Проверено`, `Ждут Проверку`, `Исправляют ошибки`

### Сотрудники
- **ФИО сотрудника** / **ФИО**
- **Филиал сотрудника** / **Филиал** — для отображения филиала

## Локальный запуск

```bash
# Любой простой HTTP-сервер
npx serve .
# или
python -m http.server 8080
```

Откройте в браузере. Для теста в Telegram используйте [@BotFather](https://t.me/botfather) → Bot Settings → Menu Button → Configure menu button → URL.
